Description

This app contains hundreds of questions and answers (continuously being updated) which are quite similar to the actual G1 test. New drivers studying using this app will definitely be prepared for the G1 test in no time.

Features
-Specially designed for new Canadian drivers living in the Ontario province
-Number question sets available for studying/review
-Audio feature for questions and answers
-Test mode: a mock testing environment
-Starred mode: this allows you bookmark or save questions that you want
-Cramming mode: this allows you quickly review the questions and answers
